# 📖 Steph — University Reading Tracker

A beautifully designed, single-file web app that helps university students track reading progress across semesters, log study sessions, and stay on top of course reading lists.

---

## ✨ Features

- **Dashboard** — Reading streak, activity heatmap, stats, and motivational quotes
- **My Library** — Add books, filter by course/semester/status, track page progress
- **Reading Session** — Built-in 25-min Pomodoro timer with session logging
- **Courses** — Organize books by course with color-coded cards
- **Goals** — Set and track targets for books, pages, or minutes read
- **Persistence** — All data saved locally via `localStorage`
- **Semester Switching** — Manage multiple semesters independently

---

## 🚀 Getting Started

No installation required. Just open the file in your browser.
```bash
git clone https://github.com/YOUR_USERNAME/Steph-reading-tracker.git
cd Steph-reading-tracker
open reading-tracker.html
```

Or simply double-click `reading-tracker.html`.
